flipnow
=========

flipnow return head or tails

## Installation
```shell
  npm install git://github.com/fariswd/flipnow.git
```
OR
```shell
  npm install flipnow
```

## Usage

```js
  var flip = require('flipnow');
  console.log(flip.flipNow());
```

## Tests
```shell
  npm test
```
